**2026-03-19**

### Added

- **API key validation in setup wizard** -- keys are validated in real-time against provider endpoints (Anthropic, Google, OpenAI, GitHub) with a debounced spinner and inline check/X badge
- **Expanded key detection** -- setup wizard now discovers API keys from `~/.config/openai/api_key` (OpenAI CLI) and `~/.anthropic/api_key` (Anthropic SDK), in addition to existing env var and settings.json sources
- **Input sanitization** -- API keys are trimmed and stripped of surrounding quotes (common copy-paste artifact) before validation
- **Build verification documentation** -- new docs page covering SBOM, GitHub build attestation, manifest signatures, code signing, and notarization verification steps
